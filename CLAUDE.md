# Koinonia Power-Up

Trello Power-Up for community fund allocation and group coordination.

## Rules

- Every feature addition or change must be logged in `updates.html` before pushing.
- Use the same dark gold style as existing entries.
- Increment the version number for each release.

## URLs

- Live: https://koinonia-bice.vercel.app
- Privacy: https://koinonia-bice.vercel.app/privacy.html
- Changelog: https://koinonia-bice.vercel.app/updates.html

## Deploy

```
git add -A && git commit -m "message" && git push
```
